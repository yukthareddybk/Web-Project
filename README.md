# Bootstrap Personal Web Project

This project is a simple personal website created using **HTML, CSS, Bootstrap, and JavaScript**.
The website demonstrates basic front-end web development concepts such as responsive navigation bars, Bootstrap components, profile cards, and JavaScript interactions.

The goal of this project is to practice building a clean and responsive web interface using **Bootstrap and basic web technologies**.

---

## 🚀 Features

* Responsive **Bootstrap navigation bar**
* **Jumbotron section** introducing the developer
* **Profile card** with image and description
* **LinkedIn button** linking to professional profile
* **GitHub button** linking to repository
* **JavaScript alert button** interaction
* Simple and clean UI design

---

## 🛠 Technologies Used

This project uses basic front-end technologies:

* **HTML5** – Webpage structure
* **CSS3** – Custom styling
* **Bootstrap 4** – Responsive UI components
* **JavaScript** – Button interaction (alert function)

## ⚙️ How the Project Works

1. The webpage loads **Bootstrap CSS and JavaScript from a CDN**.
2. A **navigation bar** is displayed at the top of the page.
3. A **jumbotron section** introduces the developer and contains a LinkedIn link.
4. A **Bootstrap card** displays profile information and a GitHub link.
5. A button triggers a **JavaScript alert message** when clicked.
